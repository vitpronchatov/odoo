Task Template & Project Template

Task Templates - Odoo Module

Project Templates - Odoo Module

This app allows your project team to create project template and task template.


Main Features:
  • This app allows you to "Create Project" from Project Template. And all Task(s) from the project template will also be copied to the new Project.
  • Reverse: This app also allows you to "Create Project Template" from Existing Project.And all Task(s) from project will also be copied to new Project Template.
  • This app allows you to "Create Task" from Task Template.
  • This app allows you to "Create Task Template" from Existing Task.
  • Also support subtasks creation from subtasks template.
  • For more details please watch the video and see the screenshots below.
